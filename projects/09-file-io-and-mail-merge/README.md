# File I/O and Mail Merge

## What this project does

A file handling project that reads names from a file and creates personalized letters from a template.

## What I practiced

- reading and writing files
- working with folders and paths
- template replacement with strings

## How to run it

```bash
python mail-merge/main.py
```

## Notes

The `read-write-files/` folder contains smaller file I/O practice examples.
