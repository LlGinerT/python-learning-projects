# Pandas Data Analysis

## What this project does

Small data analysis scripts that read CSV files and summarize information with pandas.

## What I practiced

- reading CSV files with pandas
- working with DataFrames
- exporting processed data

## How to run it

```bash
pip install -r requirements.txt
python weather_pandas.py
```

## Notes

The `squirrel-census/` folder contains a separate pandas exercise using a larger CSV file.
