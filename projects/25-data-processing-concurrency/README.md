# Data Processing Concurrency

## What this project does

A timing comparison between sequential execution, threads and processes using simulated data processing.

## What I practiced

- threading
- multiprocessing
- comparing I/O-bound and CPU-bound work

## How to run it

```bash
python main.py
```

## Notes

The Spanish `solucion.md` file is an important part of this project. It explains the results, compares the execution times and connects the code with the concurrency concepts practiced here.

The original exercise statement is also included as `enunciado.md`.
