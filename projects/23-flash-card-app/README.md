# Flash Card App

## What this project does

A flash card app for vocabulary practice that tracks known and unknown words.

## What I practiced

- Tkinter UI updates
- reading and writing CSV data
- keeping progress between sessions

## How to run it

```bash
pip install -r requirements.txt
python main.py
```

## Notes

The `data/` folder stores the vocabulary CSV files and progress file. The `images/` folder contains the card and button assets used by the Tkinter interface.
