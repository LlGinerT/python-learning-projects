# OOP Coffee Machine

## What this project does

An object-oriented version of the coffee machine project, split into classes for menu, money and machine behavior.

## What I practiced

- classes and objects
- composition
- separating responsibilities across files

## How to run it

```bash
python main.py
```

## Notes

The `examples/` folder contains earlier OOP practice code, and `docs/` keeps the original reference PDFs for the coffee machine exercise.
