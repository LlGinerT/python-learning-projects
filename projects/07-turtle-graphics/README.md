# Turtle Graphics

## What this project does

A group of small Turtle scripts that draw shapes, patterns, random walks and a simple painting-style composition.

## What I practiced

- Turtle graphics
- random colors and movement
- using loops to create drawings

## How to run it

```bash
python hist_painting.py
```

## Notes

This folder contains several independent scripts. You can run any `.py` file to see a different drawing.
