# Thread Safe Inventory

## What this project does

A multithreading example that simulates many orders updating the same inventory value.

## What I practiced

- race conditions
- locks and critical sections
- daemon threads and shared state

## How to run it

```bash
python main.py
```

## Notes

The Spanish `solucion.md` file is an important part of this project. It explains the race condition, the synchronized version and the comparison between both executions.

The original exercise statement is also included as `enunciado.md`.
